Context-specific datasets

UNHCR operates around the world, collecting data on specific populations, in specific places, and for specific purposes. There are data sources which can help you find the specific data you need.

Global Data Inventory

A searchable list of ongoing data collection, organized by UNHCR Regional Bureaux, sectors, outcome areas and annual planning.

Learn more

*(INTERNAL)

RIDL

The Raw Internal Data Library (RIDL) is UNHCR’s secure globally-supported platform for storing data collected or managed outside of UNHCR’s core data systems, including anonymized or personally identifiable operational microdata.

Learn more

*(INTERNAL)

PRIMES Dataport

PRIMES Dataport allows users to perform advanced analyses on individual case data managed through UNHCR’s proGres system, supporting identity management and protection efforts.

Learn more

*(INTERNAL)

MDL

UNHCR's Microdata Library contains microdata on forcibly displaced and stateless people, including unit-level data collected through census, registration/administrative exercises and surveys. All datasets include comprehensive metadata and supporting documents such as survey questionnaires and analytical reports.
